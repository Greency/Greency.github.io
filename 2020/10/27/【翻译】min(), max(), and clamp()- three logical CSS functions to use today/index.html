【翻译】min(), max(), and clamp()- three logical CSS functions to use today

Posted on 2020-10-27 Edited on 2020-11-02

原文地址

学习如何使用这些支持性很好的CSS函数来控制元素的大小，维持合适的间距以及完成流体布局。

随着响应式设计的发展以及变的越来越精细，CSS自身也在不断的发展并且提供给使用者更多控制权。所有的现代浏览器都支持min()，max()，clamp()这三个函数。它们是让网站和app更具动态和响应性的最新工具。

min()，max()，clamp() 可以使你的应用变的灵活以及流体排版，控制元素的改变，以及维持合适的间距。

背景

min()，max()，clamp() ，calc() 这几个数学函数允许使用加(+)，减(-)，乘(*)，除(/)四个数学表达式作为组件的值。

Safari是第一个在2019年四月推出的完整功能，在同一年的晚些时候，79版的Chromium也推出了。今年，随着75版Firefox的发布，所有常用的浏览器都只支持了min()，max()，clamp()。

例子

min()的Demo

你可以把min()，max()，clamp()用在任何有意义的CSS表达式的右边。对于min()和max()，提供一列参数的值，浏览器决定一个最小或最大的值。例如，在这个例子中：min(1rem, 50%, 10vw)，浏览器对这三个值进行计算选出最小的，并将这个值作为元素要显示的真实值。

max()的Demo

max()函数从给定的参数列表中选出最大的值。

clamp()的Demo

clamp()接受三个值：最大值，理想值（需要计算），最大值。例子：clamp(350px, 50%, 500px)

这三个CSS函数可以接受如下任何属性的值：<length>，<frequency>，<angle>，<time>，<percentage>，<number>，<integer>。你可以这样写：font-size: max(0.5vw, 50%, 2rem)，结合calc()使用：font-size: max(calc(0.5vw - 1em), 2rem) ，或者组合使用：font-size: max(min(0.5vw, 1em), 2rem)

在max()，min()，clamp()中使用calc()时，可以省略它。例如：font-size: max(calc(0.5vw - 1em), 2rem)和font-size: max(0.5vw - 1em, 2rem)是一样的。

小结

  • min(<value-list>)：从参数中选出最小的一个作为值。
  • max(<value-list>)：从参数中选出最大的一个作为值。
  • clamp(<min>, <ideal>, <max>)；在最小值和最大值范围内，就取中间的值。小于等于最小值，就取最小值；大于等于最大值，就取最大值。

完美宽度

根据Robert Bringhurst的文章The Elements of Typographic Style所说：“对于以文本大小为序列的单列页面，一行含有45到75个字符是最好的。”

浏览器会根据设置的值自动决定段落的宽度。当50%计算出的值小于45ch的时候，会取值45ch；当50%计算出的值大于75ch的时候，会取值75ch。

你可以使用min()和max()来分解clamp()。如果你想某个元素的宽度总是50%，但是由不超过75ch。你可以这样写：width: min(75ch, 50%);。本质上是通过min()设置了一个”最大值“。

同样的，你可以使用max()确保易读文字的最小值。像这样写：width: max(45ch, 50%);。浏览器会选择更大的一个，45ch或者50%，这意味着此元素至少有45ch。
